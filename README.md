### Intro ###
**3D/Paper Print Room Proof of Concept (POC).**

### How do I get set up? ###
**To build and run:**
* install Meteor if you have not already 
```
#!javascript
curl https://install.meteor.com/ | sh
```
* run 'meteor' in terminal (in the project folder)
* browse to http://localhost:3000/
